## 1. Executive Summary

This document outlines the proposed 4kW solar array installation near Sector B of Quilima Vineyards, designed to power the irrigation pump system connected to the main reservoir. The system provides sustainable energy for vineyard irrigation operations with an estimated daily output of 16-20 kWh.

## 3. Electrical Specifications

### 3.1 Solar Panels

- **Type:** Monocrystalline Silicon
- **Power Output:** 400W per panel
- **Voltage (Vmp):** ~41V DC per panel
- **Current (Imp):** ~9.76A DC per panel
- **Efficiency:** ~20-22%
- **Dimensions:** ~2000mm × 1000mm × 35mm per panel
- **Warranty:** 25 years performance

### 3.2 Inverter

- **Type:** Grid-tied DC/AC Inverter
- **Capacity:** 4.5 kW
- **Input Voltage:** 200-600V DC (string configuration)
- **Output Voltage:** 220V AC, 50Hz
- **Efficiency:** >96%
- **Protection:** IP65 rated (outdoor)
- **Monitoring:** WiFi-enabled production tracking

### 3.3 String Configuration

```
String 1: 5 panels in series → ~205V DC, 9.76A
String 2: 5 panels in series → ~205V DC, 9.76A
                        ↓
              Combined at Junction Box JB-1
                        ↓
                  4.5kW Inverter
                        ↓
                  220V AC Output → Pump Station
```

## 4. Piping System

### 4.1 Pipeline Specifications

| Parameter | Specification |
|-----------|---------------|
| Material | PVC Schedule 40 |
| Diameter | Ø 50mm (2 inches) |
| Total Length | ~360 meters |
| Burial Depth | 600mm (below frost line) |
| Pressure Rating | 150 PSI |
| Flow Rate | 40-60 L/min (estimated) |

### 4.2 Route Description

1. **Origin:** Solar array junction box (JB-1) at Sector B
2. **Path:** Southward 80m, then westward 280m
3. **Termination:** Main reservoir pump station
4. **Clearance:** Avoids existing irrigation infrastructure

### 4.3 Pump Station

- **Type:** Submersible centrifugal pump
- **Power:** 2.2 kW (220V AC)
- **Flow Rate:** 40-60 L/min
- **Head Pressure:** 30-40 meters
- **Control:** Automated pressure switch
- **Protection:** Thermal overload, dry-run protection

---

## 5. Reservoir Integration

### 5.1 Main Reservoir Specifications

| Parameter | Value |
|-----------|-------|
| Capacity | 50,000 liters |
| Type | Elevated storage tank |
| Material | Food-grade polyethylene |
| Inlet Height | 2.5 meters above ground |
| Outlet | Bottom-mounted with valve |
| Level Sensor | Float-type indicator |

### 5.2 Integration Points

- **Inlet Connection:** Top-fill from pump discharge
- **Overflow:** Side-mounted at 48,000L mark
- **Distribution:** Bottom outlet to vineyard irrigation network
- **Backup:** Grid connection available for cloudy periods
