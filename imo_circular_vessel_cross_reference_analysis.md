## 7. LESSONS LEARNED

### 7.1 Positive Findings

1. **Proactive Compliance:** Vessel invested €642,000 in circular-driven maintenance, often ahead of mandatory dates (pilot transfer, lifeboat release)
2. **Effective Inspections:** Crew discovered 3 DG documentation issues before incidents occurred
3. **Security Procedures:** BMP5 correctly implemented during piracy approach

### 7.2 Areas for Improvement

1. **Administrative Controls:** Technical fixes implemented (€642k spend) but administrative tracking weak (calibration expired anyway)
2. **Shipper Education:** Recurring IMSBC/DG documentation errors indicate external stakeholder gaps
3. **Procedure Updates:** Circular revisions not always translated to updated checklists (fumigation, CSM)

### 7.3 Root Cause Patterns

| Pattern | Frequency | Root Cause | Solution |
|---------|-----------|------------|----------|
| Shipper documentation errors | 7 incidents | External stakeholder not updated on IMO changes | Pre-qualification + education |
| Calibration/tracking failures | 2 incidents | Administrative system gaps | Digital tracking with alerts |
| Procedure not updated post-circular | 3 incidents | Circular-to-procedure translation gap | Mandatory procedure review trigger |
